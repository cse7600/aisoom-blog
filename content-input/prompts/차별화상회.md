- CTA 링크: https://chabyulhwa.vercel.app (차별화상회 가입 / 무료 견적 / 품목 조회)

SEO & 콘텐츠 규칙:
- 제목: 60자 이내, 메인 키워드 자연 포함, 검색 의도 반영 (원가/비교/도매/소분/후기)
- description: 120~160자, 메인 키워드 + 구체 수치 포함 (최소 주문 단가, 배송 조건 등)
- 분량: 본문 2,000~3,000자
- 구조: 목차 → H1 → 사장님 고민 공감(재고 부담·최소 주문량) → 서비스 차별점 H2 → 업종별 활용 사례 → 비교표 → Q&A 5개 → 정리 + CTA → 관련글 → 출처
- 구체 수치 필수: 소분 단위, 배송 단가, 평균 납품 리드타임, 결제 조건 등
- 업종 맞춤 예시 권장: 한식 밑반찬, 카페 시럽·우유, 베이커리 버터·밀가루, 배달 도시락 부자재
- 톤: 외식업 사장님과 대화하듯 현장감 있게. "~한다" 체 기본. "~습니다" 남발 금지. 은어·지나친 친근함 금지
- 금지어: delve, tapestry, leverage, robust, seamless, "살펴보겠습니다", "알아보겠습니다"
