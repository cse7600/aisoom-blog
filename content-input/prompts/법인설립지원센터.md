SEO & 콘텐츠 규칙:
- 제목: 60자 이내, 메인 키워드 + 연도(2026년) 자연 포함
- description: 120~160자, 수치·절세액·비교 포함
- 분량: 본문 2,000~3,000자
- 구조: 목차 → H1 → 문제 제기 → 세율/비용 비교표 → 절차 안내 → Q&A 5개 → 정리 + CTA → 관련글 → 출처

- 시뮬레이션 표 필수. 연매출/자본금 구간별 실제 숫자
