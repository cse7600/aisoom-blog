## **2\. 📝 키워드 조합 공식**

키워드를 조합하면 경쟁이 덜하면서 검색량이 있는 키워드를 찾을 수 있어요.

```
[지역] + [업종/장소] + [메인키워드]
```

**예시:**

* 부산 카페 CCTV 설치  
* 대구 학원 CCTV 추천  
* 서울 사무실 CCTV 가격  
* 인천 매장 CCTV 설치비용

```
[업종] + [고민/니즈] + [솔루션]
```

**예시:**

* 카페 사장님 CCTV 렌탈 vs 구매  
* 매장 운영 CCTV 설치 비용  
* 학원 CCTV 추천 후기

## **![📢][image1] \[필수\] 활동 시 문구 표기 안내**

## **DM 자동화가 뭔가요?**

릴스나 피드에 **특정 키워드로 댓글**을 달면 **자동으로 DM이 발송**되는 시스템이에요.

**예시:**

1. 릴스 캡션에 "CCTV 정보 원하시면 **'정보'** 댓글 달아주세요" 작성  
2. 팔로워가 "정보" 댓글 작성  
3. 자동으로 DM 발송 → 추천 링크 전달

**장점:**

* 댓글 하나하나 답글 안 달아도 됨  
* 24시간 자동 응대  
* 댓글 많이 달리면 → 알고리즘에 좋음 → 노출 증가

## **1\. 🛠️ 설정 방법 (ManyChat)**

### **Step 1: ManyChat 가입**

1. [manychat.com](https://manychat.com/) 접속  
2. "Get Started Free" 클릭  
3. **Instagram으로 연결** 선택  
4. 인스타그램 비즈니스 계정 연동

⚠️ **주의**: 개인 계정은 안 돼요\! 비즈니스/크리에이터 계정 필요

### **Step 2: 자동화 플로우 만들기**

1. ManyChat 대시보드 → **Automation** 클릭  
2. **\+ New Automation** 클릭  
3. **Instagram** 선택  
4. 트리거 선택: **Comments**

### **Step 3: 트리거 키워드 설정**

**Settings에서:**

* Trigger: "Comment contains keyword"  
* Keyword: `정보` (또는 원하는 키워드)  
* 대소문자 구분: OFF

**여러 키워드 설정 가능:**

* 정보  
* 링크  
* CCTV  
* 상담

### **Step 4: DM 메시지 작성**

자동 발송될 DM 내용을 작성해요.

## **6\. ⚠️ 주의사항**

### **커뮤니티 규칙 확인**

* 광고/홍보 금지 카페 많음  
* 규칙 어기면 강퇴 or 글 삭제  
* **정보 공유 형식**으로 작성해야 안전

### **하면 안 되는 것**

| ❌ 하지 마세요 | 이유 |
| ----- | ----- |
| 대놓고 광고글 | 삭제/강퇴 |
| 같은 글 여러 곳에 복붙 | 도배로 인식 |
| 링크만 달랑 올리기 | 스팸 처리 |
| 출동서비스 언급 | 키퍼 미제공 |
| 경쟁사 비교 | 에스원/KT텔레캅 직접 언급 X |

### **안전하게 활동하는 법**

1. **커뮤니티에 먼저 정착**

   * 가입 후 바로 홍보글 X  
   * 다른 글에 먼저 댓글 달며 활동  
2. **정보 공유 형식 유지**

   * "후기 공유합니다"  
   * "비교해봤습니다"  
   * "궁금하신 분 댓글 주세요"  
3. **링크는 댓글/DM으로**

   * 본문에 링크 바로 넣지 말고  
   * "관심 있으시면 공유해드릴게요"  
   * 요청한 분에게만 DM/댓글로 전달
